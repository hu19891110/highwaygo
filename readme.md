##highwaygo

安装：

 1.执行 composer install

 2.创建.env 然后加入

MAIL_FROM_ADDRESS=你的邮箱地址
MAIL_FROM_NAME=你的名字

这两个键值

3.执行php artisan migrate

大功告成！
